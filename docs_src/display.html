Display

The d-* classes allow elements to be hidden and exposed based on the device width.

The display classes apply to the device width and larger, class="d-none" will hide the element on all display widths while class="d-s-none d-m-block" will only hide the element on small devices.

visible t / hidden t
hidden t
visible t
Hidden s Only
Result Not s
Visble s Only
Result s Only

Result m+
