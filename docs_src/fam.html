Floating Action Menu

To use the tabs functionality the fusionCSS.js library needs to be included.

Horizontal Menu

<div class="fam">
	<a class="button floating accent">
		<i class="fa fa-plus"></i>
	</a>
	<ul>
		<li><a class="button floating accent mini"><i class="fa fa-pencil"></i></a></li>
		<li><a class="button floating accent mini"><i class="fa fa-send"></i></a></li>
		<li><a class="button floating accent mini"><i class="fa fa-photo"></i></a></li>
		<li><a class="button floating accent mini"><i class="fa fa-shopping-cart"></i></a></li>
	</ul>
</div>

Open On Load

The following example opens on load and the first two menu items are marked with the class "leave-open" so that when clicked they don't cause the menu to close.

<div class="fam">
	<a class="button floating accent">
		<i class="fa fa-plus"></i>
	</a>
	<ul class="exposed>
		<li><a class="button floating accent mini leave-open"><i class="fa fa-pencil"></i></a></li>
		<li><a class="button floating accent mini leave-open"><i class="fa fa-send"></i></a></li>
		<li><a class="button floating accent mini"><i class="fa fa-photo"></i></a></li>
		<li><a class="button floating accent mini"><i class="fa fa-shopping-cart"></i></a></li>
	</ul>
</div>
