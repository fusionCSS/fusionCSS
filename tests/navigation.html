Navigation

fusionCSS makes use of JavaScript to convert horizontal or vertical navigation marked with the collapseMenu class into a drop down list or if marked with SlideInMenu to a menu that slides into view on demand. This functionality requires fusionCSS.js to be included.

Horizontal Navigation

The horizontal navigation can be automatically converted to a drop down list when the browser width falls below the small device width by including the class collapseMenu e.g. <ul class="hnav collapseMenu">.

Example

<ul class="hnav">
	<li><a href="#stay">Link</a></li>
	<li class="active"><a href="#stay">Link</a></li>
	<li><a href="#stay">Link</a></li>
	<li><a href="#stay">Link</a></li>
</ul>

Mobile Navigation

fusionCSS can convert an unordered list to either a drop down menu or a slide out navigation menu.

Adding the class collapseMenu to the ul element for the menu causes a drop down menu to be created while adding the class slideInMenu generates a slide out menu from the list.

For collapseMenu, if one or more hidden classes are included then the menu will change on those steps otherwise the drop down is presented for small and tiny devices.

For a slide in menu an element of with an id of viewSlideInMenu is required, when the user clicks this element the slide in menu is displayed.

Example

<ul class="hnav pill collapseMenu slideInMenu">
	<li><a href="#stay">Menu Item 1</a></li>
	<li><a href="#stay">Menu Item 2</a></li>
	<li><a href="#stay">Menu Item 3</a></li>
</ul>

<div id="viewSlideInMenu">MENU</div>
