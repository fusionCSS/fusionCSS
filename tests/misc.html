Responsive Video

YouTube and Vimeo videos can be wrapped with <div class="flex-video"> .. </div>.

Example

<div class="flex-video">
	<iframe width="560" height="315" src="//www.youtube.com/embed/........." frameborder="0" allowfullscreen></iframe>
</div>

Scroll To Top

Adding the ID scrollToTop makes the element clickable, once clicked the page is scrolled to the top.

The default is to make the element visible once the window is scrolled past 600 pixels, this can by changed by adding show-at attribute to the element with the number of pixels of scroll after which to show the element.

Example

<div id="scrollToTop" class="hide" show-at="800">Top</div>
